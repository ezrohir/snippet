## Position
+ static 默认,元素出现在正常的流中,不受 top, bottom, left, right影响.
+ relative 移动相对定位元素,原本所占的空间不会改变.
+ fixed 元素位置相对于浏览器窗口是固定位置.
+ absolute
+ sticky
